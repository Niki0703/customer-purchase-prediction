# Customer Purchase Prediction using Machine Learning

## 📌 Overview
This project predicts whether a customer will purchase a product using machine learning.

## 🎯 Objective
To help businesses target the right customers and improve marketing efficiency.

## 📊 Dataset
- Age
- Income
- Website visits
- Previous purchases

## ⚙️ Model Used
Logistic Regression

## 📈 Results
- Accuracy: ~75%
- Output: 0 (No Purchase), 1 (Purchase)

## 🚀 Future Scope
- Use more data
- Improve accuracy with advanced models
